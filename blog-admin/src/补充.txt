1. jwt + redis

    redis做了令牌和用户信息的对应管理
        进一步增加了安全性
        登录用户做了缓存
        灵活控制用户的过期(续期,踢掉线)

3.使用完ThreadLocal之后,做了value的删除,防止了内存泄漏
4.线程安全 - update table set value = newValue where id = ~ and value = oldValue
5.线程池应用广泛,面试7个核心参数 (对当前的主业务流程无影响的操作,放入线程池执行)
